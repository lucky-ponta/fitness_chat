# <アプリ名>
Fitness　chat
## サイト概要
普段運動をする人たちが話すSNS

### サイトテーマ
アウトドア、運動

### テーマを選んだ理由
自分自身が運動好きなのもありますし、運動に限らず趣味が合う人が初めから集まってる状況を作るような機能が欲しいと思ったため（Bookersをベースに機能を付加していく）

### ターゲットユーザ
性別は男女ともに10代～30代が主

### 主な利用シーン
日々の運動成果を載せたりそれぞれの交流を持つこと
